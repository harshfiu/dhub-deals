# DHub Deals

Monorepo for the DHub Deals web platform.

## Structure

```
apps/
  DealsCustomer/   # Customer-facing deal discovery portal (Next.js)
  DealsVendor/     # Vendor portal for managing deals (Next.js)
```

## Getting Started

### Customer Portal
```bash
cd apps/DealsCustomer
npm install
npm run dev
```

### Vendor Portal
```bash
cd apps/DealsVendor
npm install
npm run dev
```
